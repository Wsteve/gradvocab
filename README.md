# GradVocab · 研究生单词阅读器

> 单页 HTML 应用，离线可用。10 篇融合科学杂志 × 篮球新闻的英语文章，覆盖 **150 个考研核心词汇**。

## 🚀 使用方式

直接用浏览器打开 `index.html`，无需安装任何东西。

```bash
open index.html
```

## 🧭 四大功能

| 功能 | 说明 |
|------|------|
| 📖 **文章阅读** | 10 篇科学×篮球英文文章，点击高亮词弹出释义+词源+拉丁诗文 |
| 🎴 **闪卡模式** | 3D 翻转卡片，键盘 Space/←→/S，筛选×排序，进度持久化 |
| 📚 **单词本** | 150 词总表，一键下载墨墨词库 / Anki CSV |
| 📋 **30 天计划** | 三阶段循序渐进，每日 checklist，自动高亮当天任务 |

## ⌨️ 快捷键（闪卡模式）

- **Space** — 翻转卡片
- **← →** — 上一张 / 下一张
- **S** — 听发音
- **H** — 回到首页（全局）
- **Esc** — 关闭弹窗（全局）

## 🛠 技术栈

纯前端，零依赖：

| 文件 | 作用 |
|------|------|
| `index.html` | 5 视图切换 + 弹窗 |
| `css/style.css` | 暗色主题，CSS 变量驱动 |
| `js/app.js` | 导航、闪卡、下载、计划、进度 |
| `js/data.js` | 150 词 + 10 篇文章 |
| `js/etymology.js` | 音节级拉丁/希腊词源 |

无 npm、无构建、无后端。浏览器直接打开即用。

## 📥 下载说明

在「📚 单词本」页面底部：
- **墨墨词库** — TSV 格式，可直接导入墨墨背单词 App
- **Anki 简单版** — CSV（单词/音标/释义）
- **Anki 详细版** — CSV（含音节、词源、所属文章）
