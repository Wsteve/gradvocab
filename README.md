# GradVocab Terminal · 英汉同步高亮版

> 单文件 HTML 应用，离线可用。10 篇融合科学杂志 × 篮球新闻的英语文章，覆盖 **150 个考研核心词汇**。

## 🚀 使用方式

直接用浏览器打开 `index.html`，无需安装任何东西。

```bash
open index.html
```

## 🧭 功能

| 功能 | 说明 |
|------|------|
| 🎴 **闪卡看板** | 3D 翻转卡片，音节点击发音，BBC/VOA 双音源，底部八国同源词 + 语境应用看板 |
| 📖 **双语阅读** | 10 篇英中对照文章，句子级 + 单词级双向同步高亮引擎 |
| 🔊 **双音源发音** | BBC 英音 / VOA 慢速美音，浏览器原生 TTS |
| 🌐 **八国同源词格** | 拉丁/法/西/俄/德/葡/意/中 同源词联动展示 |
| 💬 **取词浮窗** | 右上角 Popover，点击任意单词弹出释义+词源 |
| 📚 **150 词库** | 按章节筛选，全量内嵌 |

## 🛠 技术栈

纯前端，零依赖。单文件 `index.html`，CSS/JS 全部内联，无 npm、无构建、无后端。
